# 📈 InvestEdu-AI

고등학교 2학년인 제가 혼자 만든 프로젝트입니다.

주변을 보면 투자에 관심은 있는데 막상 어디서부터 시작해야 할지 모르는 친구들이 많았습니다. 사회초년생이나 청소년들이 투자 개념과 경험 없이 실제 돈을 잃는 경우도 적지 않고요. 그래서 "잃어도 괜찮은 환경에서 직접 경험해볼 수 있으면 어떨까?" 라는 생각으로 이 프로젝트를 시작했습니다.

---

## 어떤 프로젝트인가요?

단순히 주가를 보여주는 게임이 아니라, 실제로 주식·ETF를 매수하고 경제 상황 변화에 대응하면서 AI가 내 투자 습관을 분석해주는 **금융 교육 시뮬레이터**입니다. 브라우저만 있으면 바로 실행되고, 한 수업 시간(50분) 안에 게임 내 1년치 투자 경험을 완성할 수 있어 학교 수업에서도 바로 쓸 수 있습니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 🤖 AI 투자 코치 | 매월 포트폴리오 자동 분석 및 피드백 |
| 📊 AI 성적표 | 수익성·분산투자·리스크관리 등 6개 항목 점수화 |
| 🏆 실시간 랭킹 | Firebase 연동, 전국 학생과 순위 경쟁 |
| 📚 금융 학습 퀴즈 | PER·ETF·복리 등 10개 개념 카드 + 퀴즈 |
| ⚡ 레벨/XP 시스템 | 투자 행동마다 경험치 부여 |
| 🌐 경제 상황 시스템 | 호황·침체가 주가에 실시간 반영 |

---

## 기술 스택

순수 HTML + CSS + JavaScript 단일 파일 · Firebase Firestore · GitHub Pages

React나 Vue 같은 프레임워크 없이 만들었습니다. 누구나 파일 하나만 열면 코드를 바로 읽고 수정할 수 있어서, 오픈소스로 공개해 선생님이나 다른 학생들이 직접 커스터마이즈해서 쓸 수 있게 했습니다.

---

## 앞으로 하고 싶은 것

지금은 AI 분석이 규칙 기반으로 돌아가는데, Codex API를 연동해서 각 학생의 투자 패턴에 맞는 진짜 개인화 피드백을 주고 싶습니다. 그리고 한국을 넘어 다른 나라 학생들도 쓸 수 있도록 다국어 지원도 목표입니다.
