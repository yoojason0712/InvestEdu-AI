# InvestEdu-AI
고등학생을 위한 AI 기반 금융 교육 오픈소스 플랫폼
InvestEdu-AI는 단순한 모의투자 게임이 아닙니다. 학생이 직접 주식·ETF를 매수하고, AI가 투자 행동을 분석하며, 친구들과 실시간 랭킹으로 경쟁하는 인터랙티브 금융 교육 시뮬레이터입니다.
왜 이 프로젝트인가?
한국 고등학교 금융 교육은 여전히 이론 중심입니다. 학생들이 실제 투자를 경험할 실습 도구가 없습니다. InvestEdu-AI는 이 간극을 메우기 위해 만들어졌으며, 설치 없이 브라우저만으로 한 수업 시간(50분) 안에 1년치 투자 경험을 완성할 수 있습니다.

핵심 기능

-  AI 투자 코치 — 매월 포트폴리오 자동 분석 및 피드백
-  AI 성적표 — 수익성·분산투자·리스크관리 등 6개 항목 점수화
-  Firebase 실시간 랭킹 — 전 세계 학생과 순위 경쟁
-  금융 학습 퀴즈 — PER·ETF·복리 등 10개 개념 카드+퀴즈
- 레벨/XP 시스템 — 모든 투자 행동에 경험치 부여
- 경제 상황 시스템 — 호황·침체가 주가에 실시간 반영

기술 스택
순수 HTML + CSS + JavaScript 단일 파일 · Firebase Firestore · GitHub Pages
외부 프레임워크 없이 누구나 코드를 읽고 수정할 수 있어 오픈소스 기여 진입 장벽이 낮습니다.
Codex 지원으로 이루고 싶은 것

AI 피드백 고도화 — 규칙 기반 분석을 Codex API 기반 개인화 피드백으로 교체
맞춤형 학습 콘텐츠 생성 — 투자 행동 패턴에 따라 부족한 개념을 동적으로 추천
다국어 지원 — 한국을 넘어 글로벌 금융 교육 도구로 확장
보안 강화 — Firestore 규칙 고도화 및 미성년자 데이터 보호 정책 적용

교사도, 학생도, 개발자도 — fork 하나로 바로 쓸 수 있는 금융 교육 플랫폼.

Codex의 지원이 이 플랫폼을 더 많은 학생에게 닿게 하는 힘이 될 것입니다.
